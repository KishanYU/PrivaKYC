# Contributing to PrivaKYC

First off, thank you for considering contributing to PrivaKYC! It's people like you that make PrivaKYC such a great tool for the privacy-preserving identity space.

## 🌈 Code of Conduct
By participating in this project, you agree to abide by our code of conduct. Please be respectful and professional in all interactions.

## 🚀 How to Contribute

### 1. Report Bugs
If you find a bug, please open an issue. Include:
- A clear, descriptive title.
- Steps to reproduce the bug.
- Your environment (OS, Browser, Node version).

### 2. Suggest Enhancements
We love new ideas! If you have an idea for an enhancement, please open an issue with the "enhancement" label.

### 3. Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Issue that pull request!

## 🛠️ Style Guide
- **JavaScript**: Use ES6+ syntax. Follow the Prettier config.
- **CSS**: Use Tailwind CSS classes for styling.
- **Commits**: Follow [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) (e.g., `feat: ...`, `fix: ...`, `chore: ...`).
